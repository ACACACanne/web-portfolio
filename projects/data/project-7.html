PowerBI Dashboard

An interactive business intelligence dashboard turning raw data into actionable insights.

Category: Data Science · Stack: PowerBI, DAX, Data Modelling

View Dashboard GitHub Repo Contact Me

Story & Intent

I wanted to create a dashboard that feels intuitive and visually clean — something that empowers decision‑makers to explore data without needing technical knowledge. This dashboard highlights KPIs, trends, and relationships through interactive visuals.

Technical Breakdown

Data Modelling

I built a star schema with fact and dimension tables, ensuring efficient relationships and fast filtering.

DAX Measures

  • Total Sales
  • Year‑over‑Year Growth
  • Customer Retention
  • Average Order Value

Key Features

  • Interactive slicers
  • Drill‑through pages
  • Dynamic KPI cards
  • Trend and distribution charts

Challenges & Learnings

The challenge was designing visuals that communicate clearly without overwhelming the user. I learned how to apply BI design principles: contrast, spacing, and visual hierarchy.
