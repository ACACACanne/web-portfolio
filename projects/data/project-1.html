Rental Price Prediction

A machine learning model predicting rental prices using real‑world housing data.

Category: Data Science · Stack: Python, Pandas, Scikit‑Learn, Matplotlib

View Notebook GitHub Repo Contact Me

Story & Intent

Housing affordability is a global issue, and I wanted to explore how data can help us understand rental markets. This project predicts rental prices based on features like location, size, number of rooms, and amenities — turning raw housing data into actionable insight.

Technical Breakdown

Data Pipeline

The dataset was cleaned using Pandas, with missing values imputed and categorical variables encoded using One‑Hot Encoding. Outliers were handled using IQR filtering.

Modeling

I tested several models:

  • Linear Regression
  • Random Forest Regressor
  • Gradient Boosting Regressor

The best performance came from Gradient Boosting, achieving strong RMSE and R² scores.

Key Features

  • Full preprocessing pipeline
  • Feature importance analysis
  • Model comparison and evaluation
  • Visualisations for insights

Challenges & Learnings

The biggest challenge was dealing with multicollinearity and skewed distributions. I learned how to apply transformations and regularisation techniques to stabilise the model.
