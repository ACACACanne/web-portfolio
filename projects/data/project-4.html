Sales Forecasting

A time‑series forecasting model predicting future sales trends.

Category: Data Science · Stack: Python, Pandas, ARIMA, Prophet

View Notebook GitHub Repo Contact Me

Story & Intent

Businesses rely on accurate forecasting to plan inventory, staffing, and strategy. This project explores time‑series modelling to predict future sales using historical data.

Technical Breakdown

Data Preparation

The dataset was resampled, cleaned, and decomposed into trend, seasonality, and noise.

Models Used

  • ARIMA
  • Seasonal ARIMA
  • Facebook Prophet

Prophet performed best due to strong seasonal patterns in the dataset.

Key Features

  • Time‑series decomposition
  • Model comparison
  • Forecast visualisation
  • Error metrics (RMSE, MAE)

Challenges & Learnings

The challenge was handling irregularities and missing dates. I learned how to resample and interpolate time‑series data effectively.
