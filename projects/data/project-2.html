Customer Segmentation

Clustering customers into meaningful groups for targeted marketing.

Category: Data Science · Stack: Python, Pandas, Scikit‑Learn, K‑Means

View Notebook GitHub Repo Contact Me

Story & Intent

Businesses thrive when they understand their customers. This project uses clustering to identify distinct customer groups based on spending habits, demographics, and behavioural patterns — helping businesses personalise their strategies.

Technical Breakdown

Data Preparation

The dataset was scaled using StandardScaler to ensure fair distance calculations. PCA was used to reduce dimensionality and improve cluster separation.

Clustering

I tested several algorithms:

  • K‑Means
  • Hierarchical Clustering
  • DBSCAN

K‑Means provided the best balance of interpretability and performance.

Key Features

  • Elbow method for optimal K
  • PCA visualisation
  • Cluster profiling
  • Actionable insights for marketing

Challenges & Learnings

The challenge was ensuring clusters were meaningful, not just mathematically distinct. I learned how to validate clusters using silhouette scores and domain logic.
