A/B Testing Analysis

A statistical evaluation of experiment results to determine performance impact.

Category: Data Science · Stack: Python, SciPy, Statistics, Visualisation

View Notebook GitHub Repo Contact Me

Story & Intent

A/B testing is essential for data‑driven decision‑making. This project analyses experiment results to determine whether a new feature improves user engagement or conversion rates.

Technical Breakdown

Statistical Methods

  • Hypothesis testing
  • t‑tests and z‑tests
  • Confidence intervals
  • Effect size measurement

Key Features

  • Data cleaning and validation
  • Experiment design evaluation
  • Visualisation of group differences
  • Clear recommendation based on results

Challenges & Learnings

The challenge was ensuring the experiment was statistically sound. I learned how to check for sample bias, variance issues, and p‑value misinterpretation.
