NLP Sentiment Model

A machine learning model that classifies text sentiment using natural language processing.

Category: Data Science · Stack: Python, NLTK, Scikit‑Learn, TF‑IDF

View Notebook GitHub Repo Contact Me

Story & Intent

Language is emotional — and I wanted to explore how machine learning can interpret that emotion. This project classifies text as positive, negative, or neutral using NLP techniques.

Technical Breakdown

Preprocessing

  • Tokenisation
  • Stopword removal
  • Lemmatization
  • TF‑IDF vectorisation

Models Tested

  • Logistic Regression
  • Naive Bayes
  • SVM

SVM delivered the best accuracy and generalisation.

Key Features

  • Text preprocessing pipeline
  • Confusion matrix visualisation
  • Precision/Recall/F1 evaluation
  • Model comparison

Challenges & Learnings

The challenge was handling sarcasm and ambiguous text — a classic NLP problem. I learned how important dataset quality is for sentiment tasks.
