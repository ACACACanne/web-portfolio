Data Cleaning Pipeline

A fully automated preprocessing workflow for transforming raw data into analysis‑ready datasets.

Category: Data Science · Stack: Python, Pandas, NumPy, Regex

View Notebook GitHub Repo Contact Me

Story & Intent

Clean data is the foundation of every successful analysis. I built this pipeline to automate repetitive cleaning tasks — handling missing values, fixing inconsistent formats, and preparing datasets for modelling with minimal manual work.

Technical Breakdown

Pipeline Steps

  • Missing value detection and imputation
  • Outlier removal using IQR
  • String normalisation with Regex
  • Duplicate detection and removal
  • Feature type correction

Key Features

  • Reusable functions for any dataset
  • Automatic reporting of changes
  • Error‑proof handling of edge cases
  • Export of cleaned dataset

Challenges & Learnings

The challenge was designing a pipeline flexible enough to handle different datasets. I learned how to build modular functions that adapt to varying schemas and data types.
