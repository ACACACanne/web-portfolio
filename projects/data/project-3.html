SQL50 Challenge

Solving 50 SQL problems with clean, efficient queries and explanations.

Category: Data Science · Stack: SQL, PostgreSQL, Data Analysis

View Solutions GitHub Repo Contact Me

Story & Intent

SQL is the backbone of data analysis, and I wanted to sharpen my skills by completing the SQL50 challenge. Each problem pushed me to think critically about joins, grouping, filtering, and performance.

Technical Breakdown

Topics Covered

  • INNER, LEFT, RIGHT, FULL joins
  • Window functions
  • Aggregation and grouping
  • Subqueries and CTEs
  • Data cleaning with SQL

Approach

For each problem, I wrote a clean, readable solution and included an explanation describing the logic and reasoning behind the query.

Challenges & Learnings

The challenge was mastering window functions and optimising queries for performance. I learned how to use ranking, partitioning, and analytical functions effectively.
