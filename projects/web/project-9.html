Movie Explorer

A sleek movie search tool powered by a public API.

Category: Web Development · Stack: HTML, CSS, JavaScript, API

Live Demo GitHub Repo Contact Me

Story & Intent

I wanted to build a movie discovery tool that feels fast, responsive, and visually clean. The goal was to create a UI that makes browsing enjoyable — not overwhelming.

Technical Breakdown

Architecture

The app fetches movie data from a public API and displays results in dynamic cards. It includes search, loading states, and error handling.

Key Features

  • Live movie search
  • Dynamic movie cards
  • Poster, rating, and description display
  • Responsive grid layout

Challenges & Learnings

The challenge was handling inconsistent API data. I learned how to design UI components that adapt gracefully to missing fields.
