Interactive Quiz App

A playful, animated quiz experience built for engagement and instant feedback.

Category: Web Development · Stack: HTML, CSS, JavaScript

Live Demo GitHub Repo Contact Me

Story & Intent

This project started as a way to explore interactive UI patterns and user‑driven flows. I wanted to create something fun, lightweight, and responsive — a quiz that feels alive, with animations that guide the user through each question.

Technical Breakdown

Architecture

The quiz uses a simple JavaScript state machine to track progress, score, and transitions. Each question is rendered dynamically, and animations are triggered based on user input.

Key Features

  • Dynamic question rendering
  • Score tracking and result screen
  • Smooth animations and transitions
  • Responsive layout for mobile and desktop

Challenges & Learnings

The biggest challenge was designing a clean state flow that could handle user mistakes, backtracking, and rapid clicking. I learned how important it is to separate UI logic from game logic to keep everything predictable.
