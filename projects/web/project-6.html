Task Manager App

A clean productivity tool designed for clarity, focus, and everyday use.

Category: Web Development · Stack: HTML, CSS, JavaScript (LocalStorage)

Live Demo GitHub Repo Contact Me

Story & Intent

I wanted to build a productivity tool that feels lightweight yet reliable — something that helps users stay organised without overwhelming them. The goal was simplicity, speed, and a UI that feels calm and intentional.

Technical Breakdown

Architecture

Tasks are stored in LocalStorage, allowing the app to persist data without a backend. The UI updates dynamically as tasks are added, completed, or removed.

Key Features

  • Add, edit, and delete tasks
  • LocalStorage persistence
  • Responsive layout
  • Clean, minimal UI

Challenges & Learnings

The challenge was designing a system that updates instantly while keeping the code modular. I learned how to structure small apps cleanly using reusable functions and event delegation.
