Text Adventure Game

A story‑driven interactive game built with branching logic and dynamic UI updates.

Category: Web Development · Stack: HTML, CSS, JavaScript

Live Demo GitHub Repo Contact Me

Story & Intent

I’ve always loved interactive fiction — the kind of stories where every choice feels meaningful. This project was my way of exploring narrative design through code, building a branching adventure where the player’s decisions shape the outcome. It blends storytelling, logic, and UI design into a small but expressive interactive world.

Technical Breakdown

Architecture

The game is powered by a lightweight JavaScript engine that loads scenes from a structured data object. Each scene contains:

  • A description
  • A set of choices
  • Links to the next scenes

When the player selects a choice, the UI updates instantly, replacing the text and options without reloading the page. This creates a smooth, immersive reading experience.

Key Features

  • Branching narrative system
  • Dynamic UI updates based on player choices
  • Replayable story paths
  • Lightweight, responsive design
  • Modular scene structure for easy expansion

Challenges & Learnings

The biggest challenge was keeping the story logic readable and scalable. As the number of branches grew, I needed a clean way to structure scenes so they remained easy to edit. I learned how to design a modular narrative engine using simple data structures, making it easy to add new chapters or endings without rewriting the core logic.

“Good interactive fiction is 50% story, 50% architecture.”
