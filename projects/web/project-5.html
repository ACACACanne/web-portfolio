Recipe Finder

A clean, API‑powered recipe search tool built for speed and simplicity.

Category: Web Development · Stack: HTML, CSS, JavaScript, API

Live Demo GitHub Repo Contact Me

Story & Intent

I wanted to build a tool that makes cooking more accessible — something fast, intuitive, and visually clean. The goal was to turn recipe discovery into a smooth, enjoyable experience.

Technical Breakdown

Architecture

The app fetches recipe data from a public API and renders results dynamically. It includes loading states, error handling, and responsive card layouts.

Key Features

  • Live recipe search
  • Ingredient and dietary filters
  • Responsive recipe cards
  • API error handling

Challenges & Learnings

The biggest challenge was designing a layout that adapts to wildly different recipe data. I learned how to build flexible UI components that gracefully handle missing fields.
