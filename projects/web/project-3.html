Portfolio Website

A cinematic, modular, story‑driven portfolio built with intention and emotion.

Category: Web Development · Stack: HTML, CSS, JavaScript

Live Demo GitHub Repo Contact Me

Story & Intent

This portfolio is more than a website — it’s a digital identity. I wanted to build a platform that feels cinematic, expressive, and deeply personal, blending storytelling with technical clarity.

Technical Breakdown

Architecture

The site uses modular HTML pages, shared CSS themes, and a dynamic JavaScript loader for project cards. The design system is built around the Midnight Journal palette.

Key Features

  • Dynamic project grid powered by JavaScript
  • 3D rotating featured slider
  • Modular project pages
  • Responsive cinematic layout

Challenges & Learnings

The challenge was balancing aesthetics with performance. I learned how to structure CSS and JS to keep everything modular, reusable, and scalable.
