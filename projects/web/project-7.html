Memory Game

A playful card‑matching game built with animations and logic.

Category: Web Development · Stack: HTML, CSS, JavaScript

Live Demo GitHub Repo Contact Me

Story & Intent

I wanted to build something fun — a small game that blends logic, timing, and animation. The Memory Game became a perfect playground for experimenting with event handling and state management in JavaScript.

Technical Breakdown

Architecture

The game uses an array shuffle algorithm, click‑based event listeners, and a simple state machine to track matches, flips, and resets.

Key Features

  • Card flip animations
  • Match detection logic
  • Shuffle algorithm
  • Win state and restart button

Challenges & Learnings

The biggest challenge was preventing rapid clicking from breaking the game logic. I learned how to lock the board temporarily and manage asynchronous animations.
